## UX 및 디자인 구조 분석

### 세예의원 블로그 UX 특징
1. **블로그 포스트 형식**: 제목 + 메타 정보 (날짜, 카테고리) + 본문
2. **이미지 삽입**: Next.js Image 컴포넌트 사용, 최적화된 이미지 로딩
3. **표 형태**: 샷수별 정보를 표로 정리
4. **비디오 임베드**: YouTube 비디오 삽입
5. **관련 글 추천**: 하단에 관련 포스트 썸네일

### 오드의원 칼럼 UX 특징

2. **단계별 설명**: STEP 1, STEP 2 형태
3. **이미지와 텍스트 혼합**: 원리 설명 이미지가 글 중간에 삽입

5. **비교 설명**: 기존 울쎄라 vs 울쎄라 프로 프라임 비교

## 작동 원리
울쎄라 프로는 고강도 집속 초음파(HIFU) 기술을 사용한 리프팅 시술로, 피부 깊숙한 근막층(SMAS)까지 초음파 에너지를 전달하여 열 응고점을 만들고, 콜라겐 재생을 촉진합니다. 순간적인 조직 수축과 장기적인 콜라겐 리모델링을 통해 처짐 개선 효과를 만듭니다.

## 주요 효과
- **턱선/턱살 개선**: 이중턱과 처진 턱선을 V라인으로 정리
- **탄력 강화**: 볼살 타이트닝과 불독살 개선
- **주름 개선**: 팔자주름, 목주름 등 개선
- **피부 개선**: 피부 흡수력 향상 및 부드러워짐

## 효과 지속기간
- 평균 6개월~1년 유지되며, 샷 수와 개인의 피부 상태에 따라 달라집니다.
- 효과는 시술 직후 즉각적으로 나타나기도 하며, 2~3개월에 걸쳐 점진적으로 개선됩니다.

## 샷수별 정보
- **100샷**: 눈가, 입가 등 세밀한 부위 집중 개선, 약 6개월 유지
- **300샷**: 얼굴 전체, 얼굴 전반의 탄력 회복, 6개월~1년 유지
- **600샷**: 얼굴 + 목 등 넓은 범위, 최대 1년 이상 유지

## 기존 울쎄라와의 차이점
- 초음파 시야 범위: 최대 4.5mm → 최대 8mm
- 영상 해상도: 표준 해상도 → 향상된 해상도
- 설계 방식: DeepSee 영상 기반 → 고해상도 영상 기반, 더 맞춤화된 설계 가능

1. **블로그 형식 레이아웃**: 제목 + 본문 + 이미지 삽입
2. **표 형태 정보**: 샷수별 정보를 표로 정리
3. **비교 섹션**: 기존 vs 신형 비교 카드
4. **단계별 설명**: STEP 형태의 타임라인
5. **비디오 임베드**: YouTube 비디오 삽입
