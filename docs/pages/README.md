## 각 파일 구성

각 마크다운 파일은 다음 정보를 포함합니다:
- 검색 키워드
- 주요 콘텐츠 (텍스트)
- 이미지 참고 URL
- UX 참고 사항

## 사용 방법

이 자료는 각 페이지의 콘텐츠를 업데이트할 때 참고 자료로 활용할 수 있습니다.
